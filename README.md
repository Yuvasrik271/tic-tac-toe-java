# Tic Tac Toe Game

A simple console-based Tic Tac Toe game developed using Java.
This project demonstrates basic programming concepts such as arrays, loops, and conditional statements.

## Features

* Two-player gameplay (Player X and Player O)
* Winner detection logic
* Draw condition handling
* Input validation for moves

## How to Run

1. Compile the program:
   javac tic_tac_toe_game.java

2. Run the program:
   java tic_tac_toe_game

## How to Play

* Players take turns entering a number from 1 to 9
* Each number represents a position on the board
* Player X starts the game
* The first player to get three marks in a row (horizontal, vertical, or diagonal) wins
* If all positions are filled without a winner, the game ends in a draw
